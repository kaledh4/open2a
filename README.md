# dash3

Dashboard for open2a repo.

## Structure

- `docs/` - GitHub Pages root
  - `index.html` - Main dashboard
  - `resulted.md` - Data file (generated by cron)
- Backend runs as cron schedule (not pushed to GitHub)

## GitHub Pages

Enable GitHub Pages in repo settings:
- Source: `/docs` branch

## Cron Jobs

Backend processes run locally and update `docs/resulted.md`.
Only docs folder gets pushed to GitHub.
